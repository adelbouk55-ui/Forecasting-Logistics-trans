# Forecasting-Logistics-trans

A comprehensive REST API for **time series forecasting** and **logistics demand prediction** using multiple machine learning models (Prophet, ARIMA, LSTM, XGBoost).

## 🎯 Features

- ✅ Multiple forecasting models (Prophet, ARIMA, LSTM, XGBoost)
- ✅ REST API with Flask
- ✅ Data upload (CSV, Excel, JSON)
- ✅ Model training & evaluation
- ✅ Real-time predictions
- ✅ Performance metrics
- ✅ Comprehensive documentation
- ✅ Unit tests
- ✅ Docker support (coming soon)
